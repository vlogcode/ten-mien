Giải pháp Failover: x3 lần hoạt động ổn định

  • Bảo mật cao
  • Tốc độ cực nhanh
  • Ổn định cao
  • An toàn dữ liệu
  • Sử dụng đơn giản

  • Web Application FireWall (WAF)

    Ngăn chặn hacker và các cuộc tấn công qua lỗ hổng website.

  • Truy cập hosting được bảo mật xác thực 2 lớp (2nd layer security login)

    Bảo mật tài khoản quản trị SecureWEB hosting với xác thực token từ Google Authentication.

  • Giới hạn truy cập quản trị theo IP (Restrict Filter IP)

    Phòng ngừa rủi ro hack tài khoản quản trị bằng việc giới hạn truy cập từ đường truyền Internet cố định.

  • Giới hạn truy cập quản trị bằng xác thực 2 lớp (Restrict Authentication)

    Xác thực thông tin truy cập quản trị 2 lớp. Phòng ngừa rủi ro bị hack tài khoản quản trị.

  • Công nghệ Ultra Speed từ Kusanagi Nhật Bản

    Kusanagi Cloud Nhật Bản được đánh giá là một trong những Wordpress Cloud nhanh nhất hiện nay.

  • 80% website trên SecureWEB đạt được tốc độ load web < 1s

  • Fcache tăng tốc độ load các file tĩnh x 10 lần

  • Tuỳ chọn đa dạng version php từ 5.3 -> 7.2. Khách hàng sử dụng php 7.0 trở lên để đạt tốc độ tốt nhất

  • Hệ thống tự cân bằng tải lại đều các server (dựa trên nguyên tắc bình thông nhau)

    Các sản phẩm hosting thông thường sau 1 thời gian hoạt động sẽ bị chậm đi do dữ liệu tăng lên. Với SecureWEB hosting, hosting của khách hàng được cân bằng tải giữa các server trên hệ thống, giúp tốc độ xử lý luôn đảm bảo nhanh như lúc ban đầu.

  • Failover Solution

    Tự động chuyển hướng khi bị tấn công, hay gián đoạn

  • Multi Homing Solution

    Hạ tầng mạng ứng dụng giải pháp Multi Homing Solution. Kết hợp nhiều nguồn mạng khác nhau đảm bảo hoạt động ổn định khi một network bị gián đoạn.

  • Hight Availability Cloud

    SecureWEB được xây dựng trên nền tảng HA Cloud. Khi server vật lý gặp lỗi, Cloud tự chuyển dịch sang hệ thống server khác, giúp cho việc hoạt động an toàn và ổn định hơn.

  • Backup 3 lớp

    SecureWEB Cloud được thực hiện Backup 3 lớp nhằm đảm bảo an toàn cao về dữ liệu.

  • DR Site Backup

    Toàn bộ database được Backup DR Site sang hạ tầng Cloud tại Nhật Bản.

  • Incremental Backup

    Backup dữ liệu file hàng ngày, lưu trữ tối đa 7-14 bản trong 14 ngày.

  • Snapshot Backup
